Где кататься? Выбирать только Вам! Круто будет везде!

Но чтобы дать немного пищи для размышления сообщим краткую информацию по каждому курорту.

  • ГОРНОЛЫЖНЫЙ КУРОРТ «ГАЗПРОМ» Отлично подходит для начинающих и любителей комфортного трассового катания, а также для семейного отдыха. Здесь 23 км подготовленных трасс и есть трассы для беговых лыж!
  • ГОРНОЛЫЖНЫЙ КУРОРТ «Красная поляна» Лучше всего подходит для двух основных типов катающихся: начинающие и профи. Это обусловлено тем что трасс средней сложности достаточно мало, а имеющиеся достаточно узкие и крутые.
  • ГОРНОЛЫЖНЫЙ КУРОРТ «РОЗА ХУТОР» Это самый активный и динамичный курорт! По мнению большинства специалистов именно Роза Хутор является лучшим курортом из всех имеющихся по совокупности качеств! Здесь более 74 км подготовленных трасс самых разных уровней сложности с превосходными по своей красоте пейзажами!

На каждом курорте есть горнолыжные школы, аренда оборудования, многочисленные кафе и различная инфраструктура для отдыха!

Отличное настроение и хороший снег гарантированы!
